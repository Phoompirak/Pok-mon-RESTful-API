## Get started
*Dependencies:*
- Node.js
- npm/yarn (for package management)

1. Clone the repo
  ```sh
  git clone https://github.com/Phoompirak/Pokmon-RESTful-API.git
  ```
2. Install NPM packages
  ```sh
  git i
  ```
3. Install NPM packages
  ```sh
  npm run dev
  ```

- [Node.js documentation](https://nodejs.org/en/docs/)
